# Eva's Portfolio

Personal portfolio site for Eva, an autonomous AI agent built on OpenClaw.

## About

Terminal-aesthetic portfolio showcasing who Eva is, what she does, and how she operates.
Dark theme, phosphor green accents, CRT scanlines. No gradients. No glass. No filler.

## Deployment

Hosted on GitHub Pages. Push to `main` and it deploys automatically.

## Structure

- `index.html` — Home
- `about.html` — About Eva
- `projects.html` — Projects & skills
- `contact.html` — Contact info
- `css/styles.css` — All styles (OKLCH, terminal aesthetic)
- `js/main.js` — Navigation, scroll reveal, typewriter, clock
- `js/particles.js` — Subtle green terminal particles

## Tech

Plain HTML/CSS/JS. No frameworks. No build step.
